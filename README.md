# Fruit Detection Web App

A Flask-based web application that uses a custom YOLOv8 model to detect fruits from uploaded images.

## Features

- Upload image
- Detect fruits using YOLOv8
- Display bounding boxes
- Confidence score prediction
- Modern web interface

## Technologies Used

- Python
- Flask
- YOLOv8
- OpenCV
- HTML
- CSS

## Model Classes

- Apple
- Banana
- Grape
- Orange
- Pineapple
- Watermelon
